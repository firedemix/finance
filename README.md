# 🎓 Мои Финансы — Трекер расходов для студентов

Простое и эффективное веб-приложение для управления личными финансами, разработанное специально для студентов. Помогает понять, куда уходят деньги, планировать бюджет и контролировать лимиты.

**🌐 Живое демо:** https://finance-mlud.vercel.app/

---

## ✨ Основные функции

### 🟢 Уровень Starter
*   **Учет транзакций:** Добавление доходов и расходов (сумма, категория, дата, комментарий).
*   **История:** Список всех операций с возможностью удаления.
*   **Дашборд:** Наглядное отображение общего баланса, доходов и расходов за текущий месяц.
*   **Локальное хранение:** Данные сохраняются в `localStorage` вашего браузера.

### 🟡 Уровень Standard
*   **Визуализация:** Круговая диаграмма расходов по категориям и график активности за неделю.
*   **Бюджетирование:** Установка лимита на месяц с системой умных уведомлений.
*   **Цветовая индикация:** Полоса прогресса меняет цвет (зеленый -> желтый -> красный) при приближении к лимиту.
*   **Категории с иконками:** Удобные предустановленные категории для быстрой навигации.

### 🎁 Бонусы
*   **Темная тема:** Полноценный Dark Mode для комфортного использования в любое время суток.
*   **Экспорт данных:** Возможность скачать историю транзакций в формате CSV для анализа в Excel.
*   **Адаптивный дизайн:** Удобная работа как на ПК, так и на мобильных устройствах.

*   **Vite** — современный инструмент сборки.
*   **Tailwind CSS** — декларативная стилизация и адаптивная верстка.

*   **Lucide React** — современная библиотека иконок.
